Screenshot Touchpoints Bible

  • Screenshots product by BrowserStack is used to test cross-browser compatibility of websites.
  • Users can generate screenshots across combinations of 2000+ devices and browsers.
  • Used to compare designs.

Default UX

  • Default view

    The first time when a user lands on Browserstack Screenshots. Input and buttons will be unfocused until user starts typing.

    See design

  • Default view (URL Entered)

    User starts to type/paste a website URL. Buttons and input field are now focused.

Screenshots Generated UX

  • Screenshots Generating/Generated view

    This page shows the screenshots generated as per the OS and browser combinations. Users can share the test results(Via. Link, Slack, Asana, Email, Jira, trello and other team collaboration mediums), Download individial screen / ZIP of all screens, check the progress, cancel, run again and start a new test.
