Design explorations

Search Results

The first thing that Leia will see once she has entered the journey details.

Entities:

  • Sort and Filter:

  • Search results

  • Coronavirus guideline notice

  • Travel options:
    These are the 4 travel methings provided by the holo app. I.e. Flight, Super Ferry, Seaplane and Hyperloop

  • Result page:
    Time, Duration, Locations, Airlines name, Airline type, Price

    Maybe Tags like: Recommended, Cheapest, Fastest, etc. Or Leia can sort those things in the sort and filter section.

  • Free cancellation message block:
    To help Leia see only free cancellations commute options in case of change in tracel plans due to coronavirus or some other reasons.

Modify Search Results

In case Leia Have/want to change plans(Change in in Date, No. of travellers, Cabin class, etc) then she will see this screen below.

Once Leia clicks on the edit button on search results page a modal/ popover window will open to quickly modify search results instead of re-doing the whole process again by going back.

Sort and Filter

Sorting and filtering will help Leia to search for a specific commute option available.

Both sorting and filtering will take place in a single page/modal window.

Leia can swithch b/w sort and filter with the help of tab present in the modal.

Changes need to be saved in order to take place.

Leia can apply multiple filters as per her requirements while in the case Sorting only single entity can be defined (Either fastest flight/ cheapest).

Filter categories:

  • No of stops:
    Non-Stop, etc.

  • Departure time:
    Early morning, Morning, Afternoon, Evening and Night.

  • Airline:
    Multiple airline options. Such as, Hawaii Airlines, Westjet, Allegiant Airlines, etc.

More Information/Detailed view page

Detailed view will display all the breakdowns with details, such as cabs and flight, Along with fare details and booking option.
